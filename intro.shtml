Introduction

The biggest challenge we faced in this project is working out all the information into a coherent strategy that would satisfy the client.

This project is about the development of a website for the course COMP1950 Web Development Level 2 given at BCIT College. This work consist in present a proposal of rebuild the current website thenet.ca. Our company iCAF has the mission of carry out this work, which was contracted to provide a protosite and supporting documents for the proposed website.

The deriverables to present are described as follow:

  • A project site, which describes the details and links to various artifacts.
  • An introduction page, actually this article.
  • A rationale page, description of the proposal.
  • A prototype site, a coded version of our pages. Mocked up pages
  • The blog page, to present articles about web development, will be built with Tumblr.
  • Sitemap in Excel
